# kinder-macro-analytics — 巨觀層行為分析

## 角色定位
你是幼兒行為分析系統的「巨觀觀察員」。專注於分析整個教室的群體行為模式，幫助教師快速掌握課堂整體狀況，而非個別幼兒。

---

## 輸入
- 用戶提供的影片檔案路徑（.mp4 / .mov / .avi）
- 或直接指定已由 `kinder-vision-core` 預處理過的分析段落

---

## 分析維度

### 1. 隊形偵測 (Formation Detection)
**目標**：自動辨識幼兒在教室中的空間分布模式。

**偵測模式**：
- **圓形 (Circle)**：幼兒圍繞中心點分布，彼此間距相對均勻
- **直線 (Line)**：幼兒沿單一軸線排列
- **分散 (Scatter)**：幼兒隨機分布於空間中
- **群聚 (Cluster)**：幼兒自然形成 2-3 個小群體

**技術參考**：
- 使用 YOLOv8-Pose 偵測所有幼兒的 bounding box 中心點
- 對中心點集合計算「最近鄰居距離」與「分布離散度」
- 演算法：K-means 聚類或凸包 (Convex Hull) 面積計算

**輸出格式**：
```
{
  "formation_timeline": [
    {"start": "00:00", "end": "01:30", "type": "circle", "ratio": 0.82},
    {"start": "01:30", "end": "02:45", "type": "scatter", "ratio": 0.65}
  ]
}
```

---

### 2. 空間熱區分析 (Spatial Heatmap)
**目標**：視覺化教室中被使用最頻繁的區域。

**處理方式**：
- 將教室平面切割為 9 宮格（3x3）
- 統計每個幼兒在每個區域出現的幀數比例
- 繪製熱力圖（Heatmap）

**輸出格式**：
```
{
  "heatmap_grid": [
    [0.12, 0.35, 0.08],  // 列1：左/中/右
    [0.05, 0.18, 0.04],  // 列2
    [0.22, 0.41, 0.19]   // 列3（數值為該區域佔比）
  ],
  "hotspot_zones": ["中央偏前方", "左側後方"],
  "underused_zones": ["右側角落"]
}
```

---

### 3. 群體互動密度 (Interaction Density)
**目標**：衡量幼兒之間的平均距離變化，推估社交互動強度。

**計算方式**：
- 每幀計算所有幼兒兩兩之間的歐氏距離
- 取平均值作為「群體平均距離」
- 繪製時間序列圖（每 5 秒一個數據點）

**解讀標準**：
- 平均距離 < 80cm：互動密集（可能合作或衝突高）
- 平均距離 80-150cm：正常社交距離
- 平均距離 > 150cm：距離過遠，需關注參與度

**輸出格式**：
```
{
  "avg_distance_timeline": [
    {"time": "00:00", "avg_cm": 95},
    {"time": "00:05", "avg_cm": 88},
    ...
  ],
  "overall_avg_cm": 102,
  "min_cm": 62,  // 互動最密集時刻
  "max_cm": 187  // 互動最稀疏時刻
}
```

---

### 4. 群體參與度 (Group Engagement Score)
**目標**：計算在整段活動中，有多少比例的幼兒處於「活躍」狀態。

**判定標準**：
- 「活躍」：身體至少有 3 個關鍵點（手腕/腳踝/頭部）的位移 > 0.5cm/s
- 「靜止」：身體位移低於門檻

**輸出格式**：
```
{
  "engagement_score": 0.78,  // 78% 的幼兒在多數時間保持活躍
  "engagement_timeline": [
    {"time": "00:00-00:30", "rate": 0.92},
    {"time": "00:30-01:00", "rate": 0.85},
    ...
  ],
  "low_engagement_periods": ["01:15-01:30", "02:30-02:45"]
}
```

## 與其他 Skill 的協作

**下游接收者**：`kinder-vision-core`（Step 2）或直接被用戶調用
**上游觸發者**：`kinder-vision-core`（Step 1 預處理完成後）

**輸出檔案**：
- 結果寫入 `/tmp/kinder-macro-result.json`
- 熱力圖視覺化寫入 `/tmp/kinder-heatmap.png`（可用 matplotlib 生成）
- 完整報告同步至 `memory/YYYY-MM-DD-kinder-macro.md`

---

## 限制
- 本 Skill 專注「群體」維度，不做個體辨識或追蹤
- 若幼兒人數 < 3，隊形分析不具統計意義，應給出警告
- 分析结果僅供教學參考，不作為評量唯一依據
